This is a fanmade game based off the Infinity Castle in Demon Slayer. Requires Python 3.6-latest and Pygame installed

Movement: W, A, S, D keys

Get chased by Muzan through endless floors. Go through glowing orange portals to reach different floors. There are yellow cubes scattered wherever you go, and each yellow cube you collect gives 50, 100, or 150 Kimetsu Points at random. Reach 12000 Kimetsu Points to beat Muzan!

Have fun!
